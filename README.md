# Ink Ghostlistener

![Ink Ghostlistener](./Ink%20Ghostlistener%20バナー.jpg)

**架空のリスナーからのお便りを、コマンド一つで届けるDiscord Bot。**

配信中の演出として、ファンサーバーにお便りをランダムに投稿します。
お便りの内容はAIが生成しません。すべてオリジナルで書かれた500本以上のストックから届けます。

---

## AIを内蔵いません

Ink Ghostlistenerは **AI・LLMを一切内蔵していません。**

- お便りはすべてがオリジナルの一本一本のものです
- 現在 **500本以上のストックを用意しています**
- ラジオネームのみ、prefix と suffix の組み合わせでランダム生成します（例：「夜更かしさくら」）

500本という数は週2配信・1〜2回の使用で **約5〜6年分** に相当します。

## デモ出力

```
ラジオネーム：夜更かしさくら
配信者さんこんばんは。
今日、近所の公園で見知らぬ猫と目が合って、
なんだか時間が止まったような気がしました。
配信者さんは最近、ふと立ち止まった瞬間はありましたか？
```

---

## 特徴

- **AI・LLMの内蔵なし** — お便りの内容はすべてオリジナルのストックから取得
- **500本以上のストック** — 日常・自然・人間関係・哲学・季節・カオスの6テーマ
- **ラジオネームはランダム生成** — 毎回異なる名前で投稿（2,500通り）
- **重複防止** — 同一セッション中は同じお便りを出さない（全件消費後は自動リセット）
- **自動投稿対応** — 毎日指定した時刻に自動投稿（時刻はコマンドで変更可能）
- **サーバーオーナー限定コマンド** — ファンによるいたずら防止

---

## Ink Inc.はあなたのサーバーを見ていません

Ink Ghostlistenerは **Ink Inc.が運営するサーバーで動作します。**

ただし以下を明確にお伝えします。

- **Ink Inc.はあなたのサーバーの会話を読むことができません**
- Botは指定されたチャンネルのコマンドにのみ反応します
- 指定チャンネル以外のメッセージは一切無視します
- 読み取ったメッセージを外部に送信しません
- ユーザーの発言内容はログとして保存しません
- お便りデータはInk Inc.のサーバーからのみ読み込み、あなたのサーバーの情報は取得しません
- 処理はすべてInk Inc.のローカルサーバーで完結します

    仕組みとしては郵便屋さんと同じです。
    Ink Inc.のローカルサーバーが郵便局となり、
    あなたのポスト（Discord）にお便りを届けるだけで、
    あなたのサーバーの中を見ることはしません。

## 使い方

### 1. Botを招待する

下記リンクからあなたのファンサーバーにBotを招待してください。

**→ [Ink Ghostlistenerを招待する](https://discord.com/oauth2/authorize?client_id=1484936090766016543&permissions=67584&integration_type=0&scope=bot)**

### 2. チャンネルを登録する

お便りを投稿させたいチャンネルで、以下のコマンドを入力してください。

```
スタッフ、このチャンネルで使って
```

登録が完了すると、使えるコマンド一覧が表示されます。

### 3. コマンドを使う

登録が完了したら、そのチャンネルでコマンドを入力するだけです。

---

## コマンド一覧

すべてのコマンドは **サーバーオーナーのみ** 実行できます。

| コマンド | 動作 |
|---|---|
| `スタッフ、このチャンネルで使って` | このチャンネルをお便りの投稿先として登録する |
| `スタッフ、登録解除して` | このサーバーの設定をすべて削除する |
| `スタッフ、次のお便りちょうだい` | お便りを1件手動投稿する |
| `スタッフ、自動投稿を20:45にオンにして` | 指定時刻（JST）で自動投稿をオンにする |
| `スタッフ、自動投稿を20:45にオフにして` | 自動投稿をオフにする（時刻は記憶される） |
| `スタッフ、自動投稿の状態は` | 現在の自動投稿の状態と設定時刻を確認する |

**時刻フォーマットについて**
半角・全角の数字とコロンが混在していても正しく認識します。

```
スタッフ、自動投稿を20:45にオンにして    ✅
スタッフ、自動投稿を２０：４５にオンにして  ✅
スタッフ、自動投稿を２０:４５にオンにして   ✅
```

## お便りのテーマ一覧

500本のお便りは以下の6テーマで構成されています。

| テーマ | 本数 | 内容 |
|---|---|---|
| 日常・自然 | 100本 | 散歩・季節・身近な気づき |
| 人間関係・感情 | 100本 | 友人・家族・伝えること |
| 哲学・スピリチュアル | 100本 | 生き方・内省・価値観 |
| 季節・イベント | 100本 | 行事・季節の移ろい |
| カオス | 100本 | 日常の小さな不条理・笑えるズレ |
